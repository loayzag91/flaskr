# Flaskr
First install app requirements:

    # pip install -r requirements.txt
Pass app to to env var:

    $ export FLASK_APP=flaskr.py
Initialize database:

    $ flask initdb

Make flaskr.py executable:

    # chmod +x flaskr.py

Run app:

    # ./flaskr.py

API endpoint:

    http://localhost/blog/api/v1.0/posts/
